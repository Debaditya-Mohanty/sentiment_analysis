# sentiment_analysis
Twitter data sentiment analysis
The Sentiment Anlaysis consist of three parts .
1.Data Extraction using Twitter API and extracting the data
2.Sentiment prediction of the collected data was done using the pre-trained data of the NLTK library and TextBlob library .
3.Visualisation of the different data obtained ie User profiling ,Region profiling etc.
The data for the project is also provided here as raw data .
